# VabHub Logo设计说明

## 🎨 Logo设计

### 视觉元素

#### 图标部分
- **形状**: 圆角方形（圆角半径20px）
- **尺寸**: 120x120（可缩放）
- **渐变**: 蓝色渐变
  - 起始: #00D4FF (浅青色)
  - 中间: #0099FF (中蓝色)
  - 结束: #0066CC (深蓝色)
- **效果**: 发光光晕（青色）

#### 播放按钮
- **形状**: 三角形（右指向）
- **位置**: 图标中央
- **颜色**: 半透明白色 (rgba(255,255,255,0.3))
- **效果**: 内嵌/浮雕效果

#### 文字部分
- **字体**: "VabHub"
- **样式**: V和H大写，其余小写
- **颜色**: 深蓝色 (#0066CC) 或白色（根据主题）
- **效果**: 轻微阴影/立体效果

## 🎯 使用场景

### 1. 顶部导航栏（AppBar）
- **尺寸**: 40px
- **显示**: 仅图标

- **尺寸**: 120px
- **显示**: 图标 + 文字

- **效果**: 发光效果

- **尺寸**: 48px
- **显示**: 图标 + 文字
- **位置**: 顶部
- **功能**: 点击跳转到仪表盘

### 4. 404页面
- **尺寸**: 120px
- **显示**: 图标 + 文字
- **位置**: 页面中央

### 5. Favicon
- **格式**: SVG
- **位置**: 浏览器标签页

## 🔧 组件API

### VabHubLogo组件

```vue
<VabHubLogo 
  :size="48"           // Logo尺寸（px）
  :show-text="false"    // 是否显示文字
  :glow="true"         // 是否显示发光效果
  variant="dark"        // 主题变体：'light' | 'dark'
/>
```

### 使用示例

```vue
<!-- 基础使用 -->
<VabHubLogo />

<!-- 带文字 -->
<VabHubLogo :show-text="true" />

<!-- 大尺寸 -->
<VabHubLogo :size="120" :show-text="true" />

<!-- 浅色背景 -->
<VabHubLogo variant="light" />
```
