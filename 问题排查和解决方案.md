## 🐛 常见错误和解决

### 错误1: "npm不是内部或外部命令"

**原因**: Node.js未安装或未添加到PATH

**解决**:
1. 安装Node.js: https://nodejs.org/
2. 重启终端
3. 验证安装: `node --version`

---

### 错误2: "Cannot find module"

**原因**: 依赖未安装

**解决**:
```bash
cd frontend
npm install --legacy-peer-deps
```

---

### 错误3: "Port 5173 is already in use"

**原因**: 端口被占用

**解决**:
1. 关闭占用端口的程序
2. 或者修改端口：
   - 编辑 `frontend/vite.config.ts`
   - 修改 `port: 5173` 为其他端口（如 `5174`）

---

### 错误4: "EADDRINUSE: address already in use"

**原因**: 端口被占用

**解决**:
```powershell
# 查找占用端口的进程
netstat -ano | findstr ":5173"

# 结束进程（替换PID为实际进程ID）
taskkill /PID <进程ID> /F
```

---

### 错误5: 编译错误

**原因**: 代码有语法错误或依赖问题

**解决**:
1. 查看完整的错误信息
2. 检查是否有红色错误提示
3. 根据错误信息修复

---

## 🔧 完整重启步骤

### 步骤1: 停止所有服务

1. 关闭所有运行服务的终端窗口
2. 或者按 `Ctrl+C` 停止服务

### 步骤2: 检查端口占用

```powershell
# 检查端口8000
netstat -ano | findstr ":8000"

# 检查端口5173
netstat -ano | findstr ":5173"
```

如果有进程占用，结束它们：
```powershell
taskkill /PID <进程ID> /F
```

### 步骤3: 启动后端服务

1. 打开新的终端窗口
2. 进入后端目录：
   ```bash
   cd F:\VabHub项目\VabHub\backend
   ```
3. 启动服务：
   ```bash
   python main.py
   ```
4. 等待看到：
   ```
   INFO:     Uvicorn running on http://0.0.0.0:8000
   ```

### 步骤4: 启动前端服务

1. 打开**另一个新的**终端窗口
2. 进入前端目录：
   ```bash
   cd F:\VabHub项目\VabHub\frontend
   ```
3. 启动服务：
   ```bash
   npm run dev
   ```
4. 等待看到：
   ```
   VITE v5.x.x  ready in xxx ms
   ```

### 步骤5: 验证访问

1. 打开浏览器
2. 访问: http://localhost:5173
3. 应该能看到登录页面
