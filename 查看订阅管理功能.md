# 查看订阅管理功能 - 快速指南

## 🚀 三步查看改进后的订阅管理

### 步骤1: 运行数据库迁移（首次需要）

双击运行：
```
运行数据库迁移.bat
```

或手动运行：
```bash
cd backend
python migrate_subscription.py
```

### 步骤2: 启动后端服务

双击运行：
```
start_backend.bat
```

后端将运行在：http://localhost:8000

### 步骤3: 启动前端服务

双击运行：
```
启动前端.bat
```

前端将运行在：http://localhost:5173

## 🎯 快速测试

### 创建订阅测试
1. 点击"创建订阅"按钮
2. 填写订阅信息：
   - 标题：测试电影
   - 类型：电影
   - 选择质量：1080p
   - 选择分辨率：1080p
3. 切换到"进阶"标签页
4. 填写包含规则：`1080p|BluRay`
5. 点击"保存"

- 查看新创建的订阅卡片

## ❓ 常见问题

### 数据库迁移失败
- 确保后端服务未运行
- 检查数据库文件权限
- 查看错误信息

### 前端无法加载站点
- 确保后端服务已启动
- 检查API连接：http://localhost:8000/api/v1/sites
- 查看浏览器控制台错误

### 订阅列表为空
- 正常，首次使用没有订阅
- 点击"创建订阅"添加测试数据
