# VabHub 预览功能说明

## 🎯 预览功能

VabHub提供了多种预览方式，让您可以快速查看前端界面。

---

## 🚀 快速预览（推荐）

### 方法1: 使用快速预览脚本

**双击运行**: `快速预览.bat`

这会：
- 使用npx直接启动Vite开发服务器
- 无需安装依赖
- 支持热更新
- 自动打开 http://localhost:5173

### 方法2: 使用预览服务脚本

**双击运行**: `启动预览服务.bat`

可以选择：
1. **开发模式预览** - 支持热更新，推荐
2. **构建后预览** - 预览生产构建版本

---

## 📋 预览方式对比

### 开发模式预览 ⭐ 推荐

**优点**:
- ✅ 启动快速
- ✅ 支持热更新（修改代码自动刷新）
- ✅ 无需构建
- ✅ 完整的开发工具支持

**启动方式**:
```bash
cd frontend
npx vite --host 0.0.0.0 --port 5173
```

**访问地址**: http://localhost:5173

### 构建后预览

**优点**:
- ✅ 预览生产环境效果
- ✅ 性能优化后的版本
- ✅ 更接近实际部署效果

**启动方式**:
```bash
cd frontend
# 先构建
npx vite build
# 再预览
npx vite preview --host 0.0.0.0 --port 4173
```

**访问地址**: http://localhost:4173

## 🐛 常见问题

### 问题1: 预览无法打开

**解决**:
1. 检查服务是否启动（查看终端窗口）
2. 尝试不同的地址（localhost、127.0.0.1、IP地址）
3. 检查防火墙设置

### 问题2: npx命令失败

**解决**:
1. 检查Node.js是否安装：`node --version`
2. 检查网络连接
3. 尝试清除npm缓存：`npm cache clean --force`

### 问题3: 页面显示但API请求失败

**解决**:
1. 确认后端服务运行在 http://localhost:8000
2. 检查浏览器控制台的错误信息
3. 检查CORS配置
