## 🎨 Logo设计特点

### 视觉元素
- **图标**: 蓝色渐变圆角方形（120x120）
- **播放按钮**: 半透明白色三角形
- **文字**: "VabHub"（V和H大写）
- **发光效果**: 青色光晕

### 颜色方案
- **主色**: 蓝色渐变 (#00D4FF → #0099FF → #0066CC)
- **文字色**: 深蓝色 (#0066CC) 或白色（根据主题）
- **发光色**: 青色光晕 (rgba(0, 212, 255, 0.6))

### 动画效果
- **悬停**: 图标放大1.05倍
- **发光**: 悬停时增强发光效果
- **播放按钮**: 悬停时透明度变化

## 📋 使用方式

### 基础使用

```vue
<VabHubLogo />
```

### 带文字

```vue
<VabHubLogo :show-text="true" />
```

### 自定义尺寸

```vue
<VabHubLogo :size="120" />
```

### 不同变体

```vue
<!-- 深色背景 -->
<VabHubLogo variant="dark" />

<!-- 浅色背景 -->
<VabHubLogo variant="light" />
```

### 关闭发光效果

```vue
<VabHubLogo :glow="false" />
```
