# 最简单启动方式

## 🎯 如果所有脚本都有问题

使用这个最简单的方法，直接在命令行中运行：

---

## ✅ 方法1: 使用命令提示符（最简单）

### 步骤1: 打开命令提示符

- 按 `Win + R`
- 输入 `cmd`
- 按回车

### 步骤2: 复制粘贴以下命令

```bash
cd F:\VabHub项目\VabHub\frontend && npx vite --host 0.0.0.0 --port 5173
```

**就这么简单！** 一条命令搞定。

---

## ✅ 方法2: 使用PowerShell

### 步骤1: 打开PowerShell

- 按 `Win + X`
- 选择"Windows PowerShell"或"终端"

### 步骤2: 复制粘贴以下命令

```powershell
cd F:\VabHub项目\VabHub\frontend; npx vite --host 0.0.0.0 --port 5173
```

## ✅ 方法3: 使用批处理脚本（已修复）

## ✅ 成功标志

如果看到以下内容，说明启动成功：

```

➜  Local:   http://localhost:5173/
➜  Network: http://192.168.x.x:5173/
```

然后：
1. 打开浏览器
2. 访问 http://localhost:5173
3. 应该能看到VabHub登录页面

**重要**: 保持终端窗口打开，不要关闭！

## 🐛 如果命令失败

### 错误1: "npx不是内部或外部命令"

**解决**: Node.js未安装或未添加到PATH

1. 安装Node.js: https://nodejs.org/
2. 安装时选择"Add to PATH"
3. 重启命令提示符

### 错误2: "找不到目录"

**解决**: 检查路径是否正确

```bash
# 先检查目录是否存在
dir F:\VabHub项目\VabHub\frontend
```

### 错误3: "端口被占用"

**解决**: 使用其他端口

```bash
npx vite --host 0.0.0.0 --port 5174
```
