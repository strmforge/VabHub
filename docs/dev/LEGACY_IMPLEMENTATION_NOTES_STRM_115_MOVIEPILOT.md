# LEGACY_IMPLEMENTATION_NOTES_STRM_115_MOVIEPILOT

## 1. 115网盘集成

- 实现了115网盘API集成，支持文件列表获取、下载链接生成、字幕检测等功能
- 采用了多线程下载策略，优化大文件下载速度
- 实现了115网盘与本地媒体库的同步机制
- 支持115网盘内文件的直接播放和流媒体传输

## 2. STRM系统

- 设计并实现了完整的STRM文件管理系统，支持自动生成、更新和清理
- 支持302跳转代理模式，解决外网访问问题
- 实现了智能内外网适配，自动切换播放地址
- 支持字幕从115网盘自动检测和下载
- 实现了STRM生命周期管理，包括定期全量更新和增量同步

## 3. MoviePilot分析与对比

- 对MoviePilot架构进行了深度分析，包括URL结构、功能模块和API设计
- 对比了MoviePilot与VabHub的功能差异，确定了VabHub的差异化优势
- 分析了MoviePilot的115网盘功能实现，借鉴其优点并优化
- 研究了MoviePilot的TVDB实现，为VabHub的媒体识别提供参考

## 4. 视频字幕处理

- 实现了字幕自动检测和匹配功能
- 支持多种字幕格式（SRT、ASS等）的处理和转换
- 实现了字幕与视频的同步机制
- 支持字幕下载和本地存储管理

## 5. STRM文件操作模式

- 设计了灵活的STRM文件操作模式，支持手动和自动管理
- 实现了STRM文件的批量生成和更新
- 支持STRM文件的分类和标签管理
- 实现了STRM文件的版本控制和回滚机制
