# Manga Local Read Phase 2 用户使用指南

## 🚀 快速入门（3步开始）

**新用户？按照这3个简单步骤立即开始使用：**

### 第1步：搜索漫画
1. 在侧边栏选择"漫画" → "远程源"
2. 选择漫画源（如"动漫之家"）
3. 在搜索框中输入漫画名称，点击搜索

### 第2步：下载到本地
1. 在搜索结果中点击目标漫画封面
2. 点击"导入到本地漫画库"按钮
3. 选择下载模式（推荐"全部章节"）
4. 点击"确认"创建下载任务

### 第3步：开始阅读
1. 等待下载完成（状态变为"✅ 下载完成"）
2. 点击任务右侧的"查看本地漫画"图标
3. 使用键盘快捷键阅读：
   - **空格/↓/→**：下一页
   - **↑/←**：上一页
   - **n**：下一话，**p**：上一话

**🎉 恭喜！您已成功开始使用新的漫画阅读体验！**

## 1. 异步下载功能

### 1.1 创建下载任务

#### 步骤说明：
1. **访问远程漫画页面**
   - 在侧边栏选择"漫画" → "远程源"
   - 选择漫画源（如"动漫之家"、"哔哩哔哩漫画"等）

2. **搜索目标漫画**
   - 在搜索框中输入漫画名称
   - 点击"搜索"按钮或按回车键

3. **打开漫画详情**
   - 在搜索结果中点击目标漫画封面
   - 查看漫画详细信息（作者、状态、章节数等）

4. **创建下载任务**
   - 点击"导入到本地漫画库"按钮
   - 选择下载模式：
     - **全部章节**：下载所有章节
     - **最新N话**：只下载最新的指定数量章节
     - **选择章节**：手动选择要下载的章节
   - 点击"确认"创建任务

### 1.2 查看下载状态

#### 任务状态说明：
- **⏳ 待下载**（橙色）：任务已创建，等待处理
- **🔄 下载中**（蓝色）：正在下载，显示实时进度
- **✅ 下载完成**（绿色）：下载成功，可以开始阅读
- **❌ 下载失败**（红色）：下载失败，显示错误信息

#### 查看方式：
1. **页面头部徽章**
   - 页面右上角"下载任务"按钮会显示活跃任务数量
   - 橙色数字表示正在进行的任务数量

2. **任务对话框**
   - 点击"下载任务"按钮打开任务列表
   - 查看所有任务的详细状态和进度
   - 支持手动刷新任务状态

3. **实时更新**
   - 任务状态每5秒自动刷新
   - 下载进度条实时显示当前进度

### 1.3 任务完成后操作

#### 成功下载：
- 点击任务右侧的"查看本地漫画"图标
- 自动跳转到本地漫画库中的对应漫画
- 开始阅读已下载的章节

#### 下载失败：
- 查看错误信息了解失败原因
- 可以重新创建下载任务
- 联系技术支持解决持久性问题

## 2. 增强阅读体验

### 2.1 键盘导航

#### 基础翻页：
- **↑ / ← / PageUp**：上一页
- **↓ / → / 空格 / PageDown**：下一页
- **Home**：跳转到第一页
- **End**：跳转到最后一页

#### 章节导航：
- **n**：下一话
- **p**：上一话

#### 注意事项：
- 在输入框中输入时，键盘快捷键不会触发
- 所有导航都会平滑滚动到目标位置
- 章节切换会自动保存当前阅读进度

### 2.2 智能预加载

#### 工作原理：
- 当前页面加载完成后，自动预加载后续3页
- 预加载的图片在访问时无需等待
- 预加载失败不影响正常阅读

#### 性能优化：
- 减少阅读时的加载等待时间
- 提供流畅的连续阅读体验
- 智能内存管理，避免过度占用

### 2.3 阅读进度跟踪

#### 自动保存：
- 阅读进度每5秒自动保存一次
- 记录当前章节和页面位置
- 支持跨设备进度同步（待基础设施修复）

#### 进度恢复：
- 重新进入章节时自动恢复上次阅读位置
- 页面标题显示"上次看到：第 X 页"提示
- 支持手动跳转到任意页面

## 3. 使用场景示例

### 3.1 批量下载场景

**场景**：发现一部喜欢的漫画，想要下载所有章节

**操作流程**：
1. 在远程源中搜索漫画
2. 打开详情对话框
3. 选择"全部章节"下载模式
4. 确认创建任务
5. 在任务对话框中监控下载进度
6. 下载完成后开始阅读

**优势**：
- 一次性下载所有内容
- 后台处理，不阻塞其他操作
- 实时了解下载进度

### 3.2 追更场景

**场景**：定期追更某部漫画的最新章节

**操作流程**：
1. 在远程源中找到漫画
2. 选择"最新N话"模式（如最新3话）
3. 创建下载任务
4. 等待下载完成
5. 从上次阅读位置继续阅读

**优势**：
- 只下载需要的新内容
- 节省存储空间和下载时间
- 保持阅读连续性

### 3.3 键盘快速阅读

**场景**：使用键盘快速浏览漫画

**操作流程**：
1. 打开已下载的漫画章节
2. 使用方向键或空格键翻页
3. 使用 n/p 键快速切换章节
4. 享受流畅的阅读体验

**优势**：
- 无需鼠标操作
- 翻页速度快
- 适合长时间阅读

## 4. 故障排除

### 4.1 常见问题

#### 下载任务创建失败
**可能原因**：
- 网络连接问题
- 漫画源暂时不可用
- 服务器临时维护

**解决方案**：
1. 检查网络连接
2. 稍后重试
3. 尝试其他漫画源

#### 下载进度停滞
**可能原因**：
- 网络不稳定
- 章节内容过大
- 服务器负载过高

**解决方案**：
1. 等待一段时间观察
2. 检查网络连接
3. 重新创建下载任务

#### 键盘快捷键不响应
**可能原因**：
- 当前焦点在输入框中
- 页面未完全加载

**解决方案**：
1. 点击页面空白处确保焦点正确
2. 等待页面完全加载
3. 刷新页面重试

### 4.2 性能优化建议

#### 网络环境：
- 使用稳定的网络连接进行下载
- 避免在网络高峰期大量下载
- 考虑使用有线网络提高稳定性

#### 存储管理：
- 定期清理不需要的漫画
- 监控磁盘空间使用情况
- 避免同时下载过多大型漫画

#### 浏览器优化：
- 使用现代浏览器（Chrome/Firefox/Safari最新版本）
- 确保足够的内存空间
- 定期清理浏览器缓存
