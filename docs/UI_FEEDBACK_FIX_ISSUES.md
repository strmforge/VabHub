### #2 PlayerWall 海报卡片在小屏幕上挤得太密

- 类型：布局 / 响应式
- 描述：

  - 在小屏幕上，海报卡片之间保持适当的间距。

## 严重级别定义

- **S0**：明显 Bug（遮挡、按钮点不到、内容不可见、严重错位等）
- **S1**：读不出信息、强烈误导的 UI（比如状态颜色用反、字段标错）
- **S2**：影响日常使用但不阻塞（如间距过大、信息不突出等）
- **S3**：轻微不适但不影响使用（如颜色稍显奇怪、动画不够流畅等）
