# VabHub 前端应用

现代化、美观的WebUI设计。

## ✨ 设计特色

### 🎨 现代化设计
- **Material Design 3** - 采用最新的Material Design设计规范
- **流畅动画** - 精心设计的过渡动画和微交互
- **玻璃态效果** - 现代化的毛玻璃效果
- **渐变背景** - 精美的渐变色彩

- **明暗主题** - 支持明暗主题切换
- **自定义主题** - 可自定义主题色彩
- **自动切换** - 根据系统设置自动切换

### 📱 响应式设计
- **完美适配** - 桌面、平板、移动设备完美适配
- **触摸优化** - 移动端触摸交互优化
- **断点系统** - 完善的响应式断点

### ⚡ 性能优化
- **代码分割** - 路由级别的代码分割
- **懒加载** - 组件和图片懒加载
- **虚拟滚动** - 大数据列表虚拟滚动
- **缓存策略** - 智能的API缓存

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev

访问 http://localhost:5173

### 构建生产版本

```

- **xs**: < 600px (手机)
- **sm**: 600px - 960px (平板)
- **md**: 960px - 1264px (小桌面)
- **lg**: 1264px - 1904px (桌面)
- **xl**: > 1904px (大桌面)

## 🔧 开发指南

### 添加新页面
1. 在 `src/pages/` 创建页面组件
2. 在 `src/router/index.ts` 添加路由
3. 在 `src/layouts/components/AppDrawer.vue` 添加导航项

### 添加新组件
1. 在 `src/components/` 相应目录创建组件
2. 使用TypeScript和Composition API
3. 遵循Vuetify设计规范

### 状态管理
- 使用Pinia进行状态管理
- Store文件放在 `src/stores/`
- 按功能模块划分Store
