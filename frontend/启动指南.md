# VabHub 前端启动指南

## 🚀 快速开始

### 1. 安装依赖
```bash
cd frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173

### 3. 构建生产版本
```bash
npm run build
```

## 📁 项目结构

## 🔧 开发说明

### 添加新页面
1. 在 `src/pages/` 创建页面组件
2. 在 `src/router/index.ts` 添加路由
3. 在 `src/layouts/components/AppDrawer.vue` 添加导航项

### 添加新组件
1. 在 `src/components/` 相应目录创建组件
2. 使用TypeScript和Composition API
3. 遵循Vuetify设计规范

### 状态管理
- 使用Pinia进行状态管理
- Store文件放在 `src/stores/`
- 按功能模块划分Store

## 📝 注意事项

1. **API连接**: 确保后端服务运行在 `http://localhost:8000`
2. **环境变量**: 可在 `vite.config.ts` 中配置代理
3. **类型安全**: 所有组件使用TypeScript
4. **代码规范**: 遵循ESLint和Prettier配置
